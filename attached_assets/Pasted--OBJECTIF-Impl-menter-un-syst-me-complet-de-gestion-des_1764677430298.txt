# OBJECTIF
Implémenter un système complet de gestion des paiements multiples pour les FACTURES FOURNISSEUR uniquement (pas les dépenses)

# CONTEXTE
L'application distingue deux types de factures :
- DÉPENSES : <500k FCFA, payées immédiatement, pas de BRS → PAS de gestion paiements multiples
- FACTURES FOURNISSEUR : À régler ultérieurement, fournisseurs réguliers, avec BRS, >500k → Gestion paiements multiples

Besoin de tracker chaque paiement avec son montant, sa date et son type de règlement.

EXEMPLE D'USAGE :
- Facture fournisseur de 100 000 F
- Paiement 1 : 30 000 F le 01/12 par Wave Business
- Paiement 2 : 40 000 F le 15/12 par Espèces
- Paiement 3 : 30 000 F le 30/12 par Mobile Money
- Statut : Soldée ✅

2. Nouvelle table payments
Création :
CREATE TABLE payments (
  id SERIAL PRIMARY KEY,
  invoice_id VARCHAR NOT NULL REFERENCES invoices(id) ON DELETE CASCADE,
  amount_paid VARCHAR NOT NULL,
  payment_date VARCHAR NOT NULL,
  payment_type VARCHAR NOT NULL,
  created_at TIMESTAMP DEFAULT NOW(),
  created_by VARCHAR REFERENCES user_tokens(id)
);

CREATE INDEX idx_payments_invoice_id ON payments(invoice_id);

// Pseudo-code de validation
async function validateAddPayment(invoiceId, amountPaid, paymentDate, userTokenId) {
  // 1. Facture existe et est de type fournisseur ?
  const invoice = await db.getInvoice(invoiceId);
  if (!invoice) throw new Error("Facture introuvable");
  if (invoice.invoiceType !== "supplier_invoice") {
    throw new Error("Seules les factures fournisseur peuvent avoir des paiements multiples");
  }
  
  // 2. User autorisé (créateur de la facture) ?
  if (invoice.createdBy !== userTokenId) {
    throw new Error("Seul le créateur de la facture peut ajouter un paiement");
  }
  
  // 3. Facture déjà soldée ?
  if (invoice.paymentStatus === "paid") {
    throw new Error("Cette facture est déjà soldée");
  }
  
  // 4. Calcul reste à payer
  const paymentsResult = await db.query(`
    SELECT SUM(CAST(amount_paid AS DECIMAL)) as total 
    FROM payments 
    WHERE invoice_id = $1
  `, [invoiceId]);
  
  const totalPaid = paymentsResult.rows[0]?.total || 0;
  const amountToPay = parseFloat(invoice.amountDisplayTTC);
  const remaining = amountToPay - totalPaid;
  
  // 5. Montant valide ?
  const amount = parseFloat(amountPaid);
  if (isNaN(amount) || amount <= 0) {
    throw new Error("Le montant doit être supérieur à 0");
  }
  if (amount > remaining) {
    throw new Error(`Le montant ne peut pas dépasser ${remaining.toFixed(2)} F`);
  }
  
  // 6. Date valide ?
  const payDate = new Date(paymentDate);
  const invoiceDate = new Date(invoice.invoiceDate);
  const today = new Date();
  
  if (payDate < invoiceDate) {
    throw new Error("La date de paiement ne peut pas être antérieure à la date de la facture");
  }
  if (payDate > today) {
    throw new Error("La date de paiement ne peut pas être dans le futur");
  }
  
  // OK, créer le paiement
  return { invoice, amount, remaining };
}
Calcul du payment_status
function calculatePaymentStatus(totalPaid: number, amountToPay: number): PaymentStatus {
  const paid = parseFloat(totalPaid.toFixed(2));
  const toPay = parseFloat(amountToPay.toFixed(2));
  
  if (paid === 0) return "unpaid";
  if (paid >= toPay) return "paid";
  return "partial";
}
