## FORMAT ZOHO BILLS - 29 COLONNES

### Headers CSV (ordre exact)
Bill Date,Bill Number,PurchaseOrder,Bill Status,Vendor Name,Due Date,Currency Code,Exchange Rate,Account,Description,Quantity,Rate,Tax Name,Tax Percentage,Is Inclusive Tax,Tax Type,Vendor Notes,Terms & Conditions,Customer Name,Project Name,Item Type,Adjustment,Purchase Order Number,Is Discount Before Tax,Entity Discount Amount,Discount Account,Is Landed Cost,Warehouse Name,Branch Name

### Mapping colonnes (synthèse)

| Colonne | Valeur | Notes |
|---------|--------|-------|
| Bill Date | `date` (YYYY-MM-DD) | Date facture |
| Bill Number | `invoice_number` | Ex: FAC-2024-001 |
| Bill Status | "Open" | Toujours |
| Vendor Name | `suppliers.name` | Fournisseur |
| Due Date | `date` | Même que Bill Date |
| Currency Code | "XOF" | Fixe |
| Exchange Rate | 1 | Fixe |
| Account | `categories.account_name` | Compte comptable |
| Description | `description` | Description facture |
| Quantity | 1 | Fixe (1 ligne par facture) |
| Rate | `amount_real_ttc` | Montant TTC réel |
| Tax Name | "TVA" si `vat_applicable`, vide sinon | Conditionnel |
| Tax Percentage | 18 si `vat_applicable`, vide sinon | Conditionnel |
| Is Inclusive Tax | TRUE si `vat_applicable`, vide sinon | Conditionnel |
| Tax Type | "ItemAmount" si `vat_applicable`, vide sinon | Conditionnel |
| Project Name | `projects.name` | Nom projet |
| Item Type | "goods" | Fixe |
| Adjustment | 0 | Fixe |
| Is Discount Before Tax | FALSE | Fixe |
| Entity Discount Amount | 0 | Fixe |
| Is Landed Cost | FALSE | Fixe |
| **Autres colonnes** | Vide | Vendor Notes, Terms, Customer Name, PO Number, Discount Account, Warehouse, Branch |

**DIFFÉRENCE CLÉ avec Expenses :** Pas de colonne "Paid Through" (mode paiement non exporté pour Bills).

## BACKEND - ROUTE EXPORT

### Route GET /api/admin/export-zoho-bills

**Query params :**
- `user` : 'michael' | 'marine' | 'fatou' | 'all'
- `date_start` : YYYY-MM-DD (optionnel)
- `date_end` : YYYY-MM-DD (optionnel)

**Exemple :**
GET /api/admin/export-zoho-bills?user=michael&date_start=2024-11-01&date_end=2024-11-30

**Cohérence avec export Dépenses existant :**
- `/api/admin/export-zoho-expenses` (existant)
- `/api/admin/export-zoho-bills` (nouveau)

### Logique backend

**Réutiliser la structure de l'export Dépenses** en adaptant :

**Requête SQL :**
SELECT ...
FROM invoices
JOIN categories ON category_id = categories.id
JOIN suppliers ON supplier_id = suppliers.id
JOIN projects ON project_id = projects.id
WHERE invoice_type = 'supplier_invoice'
[+ filtres user et dates]
ORDER BY date ASC

**Note :** Pas de LEFT JOIN payment_methods_mapping (pas utilisé pour Bills).

## RÈGLES DE FILTRAGE

**Critère base :**
WHERE invoice_type = 'supplier_invoice'

**Filtrage user :**
- Michael → `user_id = [ID Michael]`
- Marine → `user_id = [ID Marine]`
- Fatou → `user_id = [ID Fatou]`
- Toutes → tous users

**Filtrage dates :**
- Si date_start → `date >= date_start`
- Si date_end → `date <= date_end`
- Si vides → toutes les factures

**Tri :** `ORDER BY date ASC`
