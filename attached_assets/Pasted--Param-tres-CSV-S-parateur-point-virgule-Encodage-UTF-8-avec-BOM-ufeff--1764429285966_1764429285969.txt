**Paramètres CSV :**
- Séparateur : **point-virgule** (;)
- Encodage : **UTF-8 avec BOM** (\ufeff au début)
- Format date : **DD/MM/YYYY** (ex: 29/11/2025)
- Format montant : **décimales avec point** (ex: 315000.00)
- Guillemets : autour des valeurs texte

## FILTRAGE

**NE PAS ajouter de filtre `invoice_type`**

Les exports Axonaut incluent TOUTES les factures (Dépenses + Factures Fournisseurs).

Garder le comportement actuel :
- Export Michael → WHERE `user_id = Michael` (tous types)
- Export Marine → WHERE `user_id = Marine` (tous types)
- Export Fatou → WHERE `user_id = Fatou` (tous types)

## CALCULS MONTANTS

### Si vat_applicable = true (TVA 18%)

**Montant HT :**
`amount_display_ttc / 1.18`

**Montant TVA 18% :**
`amount_display_ttc - (amount_display_ttc / 1.18)`

**Montant TTC :**
`amount_display_ttc`

### Si vat_applicable = false (pas de TVA)

**Montant HT :**
`amount_display_ttc` (HT = TTC car pas de TVA)

**Montant TVA 18% :**
0

**Montant TTC :**
`amount_display_ttc`
