# REMPLACER MODE "ESPÈCES" PAR "WAVE/ESPÈCES DE FATOU" POUR FATOU

## CONTEXTE
Pour Fatou uniquement, le mode de paiement "Espèces" doit être remplacé par "Wave/Espèces de Fatou" qui sera mappé vers "Caisse - Fatou" dans les exports Zoho.

## OBJECTIF
1. Ajouter le nouveau mapping en BDD
2. Remplacer "Espèces" par "Wave/Espèces de Fatou" dans l'interface pour Fatou
3. Optionnel : Mettre à jour les anciennes factures de Fatou

PARTIE 3 : METTRE À JOUR LES ANCIENNES FACTURES (OPTIONNEL)
Si tu veux que les anciennes factures de Fatou avec "Espèces" affichent "Caisse - Fatou" dans les exports :
PROD
-- Identifier l'ID de Fatou
SELECT id, name FROM users WHERE name LIKE '%Fatou%';

-- Mettre à jour les factures de Fatou
UPDATE invoices
SET payment_type = 'Wave/Espèces de Fatou',
    updated_at = NOW()
WHERE payment_type = 'Espèces'
AND user_id = (SELECT id FROM users WHERE name LIKE '%Fatou%');
Vérification
-- Voir les factures mises à jour
SELECT 
  u.name as user_name,
  i.invoice_number,
  i.payment_type,
  i.date
FROM invoices i
JOIN users u ON i.user_id = u.id
WHERE u.name LIKE '%Fatou%'
AND i.payment_type = 'Wave/Espèces de Fatou'
ORDER BY i.date DESC
LIMIT 10;
Note : Cette étape est optionnelle. Si tu ne la fais pas, les anciennes factures de Fatou avec "Espèces" continueront à afficher "Caisse en monnaie nationale" dans les exports.
