# TABLE MAPPING MODES DE PAIEMENT VERS ZOHO

## OBJECTIF
Créer une table de mapping pour convertir les modes de paiement Filtreplante vers les noms exacts des comptes Zoho Books.

## CONTEXTE
Pour les exports Zoho (Dépenses et Factures Fournisseurs), les noms de comptes bancaires doivent correspondre exactement aux noms dans Zoho. Cette table centralise ces équivalences.

## CRÉATION TABLE

### Nom de la table : payment_methods_mapping

### Colonnes :

| Colonne | Type | Contraintes | Description |
|---------|------|-------------|-------------|
| id | integer | PRIMARY KEY, AUTO_INCREMENT | Identifiant unique |
| app_name | varchar(100) | NOT NULL, UNIQUE | Nom du mode de paiement dans l'app Filtreplante |
| zoho_name | varchar(150) | NOT NULL | Nom exact du compte dans Zoho Books |
| created_at | timestamp | DEFAULT NOW() | Date de création |
| updated_at | timestamp | DEFAULT NOW() | Date de modification |

### Indexes

Créer un index sur `app_name` pour optimiser les recherches lors des exports.

## DONNÉES À INSÉRER

Insérer les 5 mappings suivants :

| app_name | zoho_name |
|----------|-----------|
| Wave Business | Banques - autres (Wave Business Principal) |
| Wave Business Caisse | Banques - autres (Wave Business Caisse) |
| Espèces | Caisse en monnaie nationale |
| Perso remboursé | Banques - autres (Wave Business Principal) |
| Carte bancaire | Banques en monnaie nationale |

**IMPORTANT :** Respecter exactement les majuscules, minuscules, espaces et accents dans `zoho_name` (sensible à la casse pour Zoho).

## UTILISATION FUTURE

Cette table sera utilisée par :
1. Export Dépenses Zoho (à venir)
2. Export Factures Fournisseurs Zoho (à venir)

Lors d'un export, le système fera une jointure ou une requête pour récupérer le `zoho_name` correspondant au `payment_method` de la facture.
