CORRECTION 3 : ORDRE AFFICHAGE CATÉGORIES
Dans le dropdown catégories, afficher dans cet ordre de priorité :
En premier : "Achats de matières premières et fournitures" (account_code '6021000000')
En deuxième : "Prestation de services" (account_name contient "prestations de services")
Ensuite : Toutes les autres par ordre alphabétique
Code exemple :
const sortedCategories = [...categories].sort((a, b) => {
  // Priorité 1 : Achats de matières premières
  if ((a.account_code || a.accountCode) === '6021000000') return -1;
  if ((b.account_code || b.accountCode) === '6021000000') return 1;
  
  // Priorité 2 : Prestation de services
  const aIsPrestation = (a.account_name || a.accountName)?.includes("prestations de services");
  const bIsPrestation = (b.account_name || b.accountName)?.includes("prestations de services");
  if (aIsPrestation) return -1;
  if (bIsPrestation) return 1;
  
  // Reste : ordre alphabétique
  const aName = a.app_name || a.appName;
  const bName = b.app_name || b.appName;
  return aName.localeCompare(bName);
});
