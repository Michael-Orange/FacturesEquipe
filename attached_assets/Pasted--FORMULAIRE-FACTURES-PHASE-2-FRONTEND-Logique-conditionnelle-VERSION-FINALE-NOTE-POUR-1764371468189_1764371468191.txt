## EXEMPLE CONCRET LOGIQUE BRS (100 FCFA)

**BRS = OUI :**
- Utilisateur saisit : 100 FCFA (montant payé après retenue)
- Montant réel comptable : 100 / 0.95 = 105 FCFA (avant retenue)
- Affichage : MONTRER les 2 montants car différents

**BRS = NON :**
- Utilisateur saisit : 100 FCFA
- Montant réel : 100 FCFA (identique)
- Affichage : NE PAS MONTRER de champ supplémentaire car identiques

useEffect(() => {
  if (!selectedCategory) return;
  
  const accountName = selectedCategory.account_name || selectedCategory.accountName;
  const isPrestationServices = accountName === "Achats d'études et prestations de services";
  
  if (!vatApplicable && isPrestationServices) {
    setShowBRSField(true);
  } else {
    setShowBRSField(false);
    setHasBRS(false);
    setAmountRealTTC(null);
    setShowRealAmountField(false);
  }
}, [vatApplicable, selectedCategory]);
useEffect 5 : Calcul Montant TTC réel (BRS) - LOGIQUE CORRIGÉE
useEffect(() => {
  if (!showBRSField || !amountDisplayTTC || amountDisplayTTC <= 0) {
    setAmountRealTTC(null);
    setShowRealAmountField(false);
    return;
  }
  
  if (hasBRS) {
    // BRS = Oui : utilisateur a saisi montant APRÈS retenue
    // Montant réel comptable (avant retenue) = montant saisi / 0.95
    // Exemple : 100 / 0.95 = 105 FCFA
    const realTTC = parseFloat(amountDisplayTTC) / 0.95;
    setAmountRealTTC(realTTC);
    setShowRealAmountField(true); // AFFICHER car montants différents
  } else {
    // BRS = Non : pas de retenue
    // Montant réel = montant affiché
    setAmountRealTTC(parseFloat(amountDisplayTTC));
    setShowRealAmountField(false); // NE PAS AFFICHER car identiques
  }
}, [hasBRS, showBRSField, amountDisplayTTC]);
useEffect 6 : Forcing Type de facture
useEffect(() => {
  const amount = parseFloat(amountDisplayTTC) || 0;
  const accountName = selectedCategory?.account_name || selectedCategory?.accountName;

    });
    
    if (!response.ok) {

    }
    
    const result = await response.json();
    alert('Facture soumise avec succès !');

    console.error('Erreur soumission:', error);

  }
