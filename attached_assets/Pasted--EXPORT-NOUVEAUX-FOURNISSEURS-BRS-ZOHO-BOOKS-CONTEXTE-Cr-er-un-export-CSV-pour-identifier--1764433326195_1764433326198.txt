# EXPORT NOUVEAUX FOURNISSEURS BRS - ZOHO BOOKS

## CONTEXTE
Créer un export CSV pour identifier les fournisseurs créés pendant une période donnée et ayant des factures avec BRS (Retenue à la source 5%).

## OBJECTIF
Export simple listant les nouveaux fournisseurs BRS avec 3 colonnes :
1. Nom du fournisseur
2. Date de création du fournisseur
3. Nombre de factures BRS sur la période

## FORMAT EXPORT - 3 COLONNES

### Headers CSV
Nom Fournisseur,Date Création,Nombre Factures BRS

### Colonnes

| N° | Colonne | Source | Format |
|----|---------|--------|--------|
| 1 | Nom Fournisseur | `suppliers.name` | Texte |
| 2 | Date Création | `suppliers.created_at` | DD/MM/YYYY |
| 3 | Nombre Factures BRS | COUNT(invoices) | Nombre entier |

### Paramètres CSV

- **Séparateur :** virgule (,)
- **Encodage :** UTF-8 avec BOM
- **Format date :** DD/MM/YYYY (ex: 15/11/2024)

Requête SQL
Logique :
Récupérer les fournisseurs qui ont :
suppliers.created_at >= date_start (créés pendant ou après le début de la période)
Au moins 1 facture avec has_brs = true entre date_start et date_end
Type facture = 'supplier_invoice' (Factures Fournisseurs uniquement)
Requête SQL :
SELECT 
  suppliers.name AS nom_fournisseur,
  suppliers.created_at AS date_creation,
  COUNT(invoices.id) AS nombre_factures_brs
FROM suppliers
JOIN invoices ON invoices.supplier_id = suppliers.id
WHERE suppliers.created_at >= date_start
  AND invoices.invoice_type = 'supplier_invoice'
  AND invoices.has_brs = true
  AND invoices.date >= date_start
  AND invoices.date <= date_end
GROUP BY suppliers.id, suppliers.name, suppliers.created_at
ORDER BY suppliers.created_at ASC
Notes :
GROUP BY pour compter les factures par fournisseur
Filtrer sur suppliers.created_at >= date_start (fournisseurs créés pendant la période)
Filtrer sur invoices.has_brs = true (uniquement factures BRS)
Filtrer sur dates factures entre date_start et date_end
Génération CSV
Mapper chaque ligne :
Nom Fournisseur → suppliers.name
Date Création → formater suppliers.created_at en DD/MM/YYYY
Nombre Factures BRS → COUNT(invoices.id)
Headers + données
Encodage UTF-8 BOM
Séparateur virgule
Nom fichier
Format : Nouveaux_Fournisseurs_BRS_YYYYMM.csv
Où YYYYMM = année + mois de la date de fin.
Exemple : Nouveaux_Fournisseurs_BRS_202411.csv
